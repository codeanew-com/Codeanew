The primary gradient always flows left to right, warm to cool. Used in the logomark, hero elements, and key visual accents only. Never apply to body text.
Amber Gold — When to use
  • Primary CTA buttons
  • Active states & highlights
  • Section label accents
  • Logomark gradient (start)
  • Important numbers & stats
Teal / Blue — When to use
  • Logomark gradient (end)
  • Links & interactive elements
  • Code blocks & technical UI
  • Secondary badges & tags
  • Informational callouts
Deep Navy — When to use
  • Primary backgrounds (dark mode)
  • Headings on light backgrounds
  • Navbar & footer
  • Code backgrounds
  • Dark UI cards
Off White — When to use
  • Page backgrounds (light mode)
  • Section alternating backgrounds
  • Form field backgrounds
  • Card backgrounds (secondary)
  • Print backgrounds

codeanew speaks with confidence and clarity. Our voice is direct and knowledgeable without being cold or overly technical. We balance professionalism with warmth — like a brilliant colleague, not a corporate manual.
